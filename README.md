# ECON 5200 — Replication Study

**Card, D., & Krueger, A. B. (1994)**  
*Minimum Wages and Employment: A Case Study of the Fast-Food Industry in New Jersey and Pennsylvania.*

This project replicates one of the most influential empirical papers in labor economics, which challenged the traditional prediction that minimum wage increases reduce employment.
